# Control Flow

Control flow is the order in which individual statements, instructions, or function calls are executed or evaluated. It identifies the "path" a program takes.

## 1. Decision Making (Conditionals)

Conditionals allow a program to perform different actions based on whether a condition is **True** or **False**.

- **If / Else**: The "Fork in the Road". If the condition is met, do A; otherwise, do B.
- **Switch / Match**: The "Checklist". Compare a single value against multiple possible cases.

## 2. Repetition (Loops)

Loops allow you to execute a block of code multiple times without rewriting it.

- **Definite Loops (For)**: Used when you know exactly how many times you want to repeat (e.g., "Do this for every item in a list").
- **Indefinite Loops (While)**: Used when you want to repeat until a specific condition changes (e.g., "Keep reading data until the file ends").

## 3. Interruptions

- **Break**: Stop everything and exit the loop immediately.
- **Continue**: Skip the rest of the current turn and jump to the next one.
